# Gravity Shift

A physics-based puzzle game where you manipulate gravity to guide a ball through mazes.

## Features

- 10 progressive levels (Easy to Expert)
- Physics simulation with gravity control
- Star collection system
- Lives tracking
- Clean gray and black design

## Tech Stack

- React
- Vite
- Canvas API
- Custom Physics Engine

## Run Locally

```bash
npm install
npm run dev
```

## How to Play

- Use arrow buttons to change gravity direction
- Guide the ball to the goal
- Collect stars for bonus points
- Complete all 10 levels

## Build

```bash
npm run build
```

## Deploy

Deploy the `dist` folder to any static hosting service (Vercel, Netlify, GitHub Pages, etc.)
